# The SEO checklist this engine ships by default

You should not need a plugin for the basics.

## Built-in

- Article and Organization schema.org JSON-LD on every post
- BreadcrumbList JSON-LD
- FAQPage JSON-LD when `faqItems` is set in frontmatter
- Open Graph and Twitter Card tags
- Canonical URLs (auto, override per post)
- `noindex` toggle per post
- XML sitemap (via `@astrojs/sitemap`)
- RSS feed
- robots.txt

## You still own

- Internal linking strategy
- Submitting the sitemap to Search Console / Bing
- Writing decent meta titles and descriptions
- Image optimisation and alt text
