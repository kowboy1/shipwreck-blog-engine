# Hello world

This is the first MDX post on the NitroBlog AI.

The frontmatter above is validated by Zod at build time — bad fields fail the build with a clear error.

## What you get

- A static blog mounted at `/blog` on any host site
- Full SEO out of the box: schema.org, sitemap, RSS, OG, canonical
- Theme tokens that match the host site's brand
- Engine updates flow via `npm update`
